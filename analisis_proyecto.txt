# ANÁLISIS DEL PROYECTO GENESISCLUB

## 📋 DESCRIPCIÓN GENERAL
GenesisClub es una aplicación web completa para la gestión de un club deportivo/social, desarrollada con arquitectura full-stack. El sistema permite gestionar miembros (socios y jugadores), solicitudes de ingreso, rubros/categorías y actividades del club.

## 🏗️ ARQUITECTURA TÉCNICA

### Backend (genesisClub-back-end)
- **Framework:** Spring Boot 3.5.13
- **Lenguaje:** Java 21
- **Base de datos:** MySQL 8.0.33
- **Seguridad:** Spring Security con JWT (Nimbus JOSE JWT 10.6)
- **ORM:** Spring Data JPA
- **Email:** Spring Boot Mail
- **Validación:** Bean Validation
- **Plantillas:** Thymeleaf

### Frontend (genesisclub-fond-end)
- **Framework:** Angular 20.0.0
- **Lenguaje:** TypeScript 5.8.2
- **Autenticación:** Auth0 Angular JWT 5.2.0
- **Programación reactiva:** RxJS 7.8.0
- **Formularios:** Reactive Forms

## 🔒 SEGURIDAD (CRÍTICO)
Según análisis de seguridad realizado:
- ⚠️ 37 vulnerabilidades encontradas
- 🔴 14 críticas, 7 altas, 14 medias, 2 bajas
- Problemas principales:
  - Credenciales en texto plano
  - Configuraciones inseguras en producción
  - Endpoints públicos sin protección
  - CORS wildcard
  - Logout inefectivo
