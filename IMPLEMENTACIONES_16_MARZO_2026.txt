8. FLUJO DE USO FINAL
================================================================================

PARA ADMINISTRADOR (Ver Socio):
  1. Accede a /socio/[id]
  2. Ve rubros asociados en sección "Rubros Asociados"
  3. Puede agregar rubros con selector dropdown
  4. Puede desasociar rubros con botón "Desasociarme"

PARA SOCIO (Panel Socio):
  1. Acedera a /socio/rubros
  2. Ingresa clave de acceso en el formulario
  3. El sistema busca el rubro por clave
  4. Si es válido, se asocia automáticamente
  5. Aparece en "Mis Rubros Asociados"
  6. Puede solicitar acceso a otros rubros
  7. Puede desasociarse en cualquier momento

10. PRUEBAS RECOMENDADAS
================================================================================

BACKEND:
  □ POST /api/rubro-socio/socio/{idSocio}/clave/{claveAcceso}
    - Clave válida → asociación exitosa
    - Clave inválida → error 400
    - Clave inactiva → error 400
    - Socio no existe → error 404
    - Ya asociado → error 400

  □ GET /api/rubro-socio/socio/{idSocio}
    - Retorna lista de rubros asociados

  □ DELETE /api/rubro-socio/rubro/{idRubro}/socio/{idSocio}
    - Desasocia correctamente

FRONTEND:
  □ /socio/rubros
    - Carga rubros disponibles
    - Ingresa clave y se asocia
    - Clave inválida → muestra error
    - Desasociación funciona

  □ /admin/ver-socio/{id}
    - Muestra rubros asociados
    - Selector excluye asociados
    - Asociar nuevo rubro funciona
    - Desasociar funciona
